# Streamlit Tensile Testing Application

This repository contains a Streamlit application designed for tensile testing data analysis. Below are detailed steps on how to install and run the application.

## Requirements
To run this application, make sure you have the following software installed:
- Python 3.7 or higher
- pip (Python package installer)

## Installation Instructions
1. **Clone the Repository**  
   Clone this repository to your local machine using the following command:
   ```bash
   git clone https://github.com/davidag96-boop/CTM_Practica1.git
   ```

2. **Navigate to the Project Directory**  
   Change to the project directory:
   ```bash
   cd CTM_Practica1
   ```

3. **Create a Virtual Environment** (optional but recommended)  
   Create and activate a virtual environment:
   ```bash
   python -m venv venv  
   source venv/bin/activate  # For Linux or macOS  
   venv\Scripts\activate  # For Windows
   ```

4. **Install Required Packages**  
   Install necessary Python packages using pip:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Application
To run the Streamlit application, use the following command:
```bash
streamlit run app.py
```
Open your web browser and navigate to `http://localhost:8501` to view the application.

## Usage
1. Upload your tensile testing data in .csv format.
2. Follow the on-screen instructions to analyze the data.
3. Visualizations of the tensile tests will be shown on the application.
