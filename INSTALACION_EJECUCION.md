# Guía de Instalación y Ejecución para la Aplicación Streamlit de Ensayo de Tensión

## Instalación

### Requisitos Previos
- Asegúrate de tener Python instalado (preferiblemente Python 3.7 o superior).
- Necesitarás pip para instalar dependencias.

### Instalación en Windows
1. Abre PowerShell o el símbolo del sistema.
2. Clona el repositorio:
   ```bash
   git clone https://github.com/davidag96-boop/CTM_Practica1.git
   ```
3. Navega al directorio del proyecto:
   ```bash
   cd CTM_Practica1
   ```
4. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Ejecución
Para ejecutar la aplicación, utiliza el siguiente comando desde el directorio del proyecto:
```bash
streamlit run app.py
```

## Sección de Solución de Problemas
- **Error de dependencias**: Asegúrate de que todas las dependencias estén instaladas correctamente. Revisa `requirements.txt`.
- **Problemas con la ejecución**: Verifica que estés en el directorio correcto y que estés utilizando la versión adecuada de Python.

## Referencias al estándar ISO 6892-1
- ISO 6892-1: Prueba de tensión de materiales a temperatura ambiente: Método de ensayo para materiales metálicos. Consultar la norma para procedimientos y requisitos específicos.
